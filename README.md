# Bloomington Counseling & Wellness

Fast, static-first Next.js site for a private practice Licensed Clinical Social Worker in Bloomington, Indiana. Built with the App Router, TypeScript, and Tailwind CSS for Cloudflare Pages deployment.

## Getting started

```bash
npm install
npm run dev
```

## Commands

- `npm run dev` – start the local dev server
- `npm run lint` – run ESLint
- `npm run build` – static export build (Cloudflare Pages friendly)
