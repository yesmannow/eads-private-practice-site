# Content Files Directory

This directory contains all the editable content for your website, managed through Pages CMS.

## File Structure

- `site-settings.json` - Global site configuration (name, contact info, etc.)
- `navigation.json` - Navigation menu items
- `homepage.json` - Homepage content (hero, features, etc.)
- `about-page.json` - About page content
- `fees-policies.json` - Fees and policy information
- `faqs/` - Collection of FAQ items (one file per FAQ)
- `specialties/` - Collection of service/specialty descriptions
- `resources/` - Collection of blog posts or resource articles

## Editing Content

**Do NOT edit these files directly!** Use Pages CMS at https://app.pagescms.org to edit content through the web interface.

The `.pages.yml` file in the repository root defines the structure and validation rules for all content.

## File Formats

All content files use JSON format for easy editing and parsing. The Pages CMS interface will handle formatting automatically.
